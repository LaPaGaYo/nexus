# Design Consultation Cheatsheet

Use this reference when `/design-consultation` needs fast option generation without bloating the active skill.

## Direction Palette

- Aesthetics: brutally minimal, maximalist chaos, retro-futuristic, luxury/refined, playful/toy-like, editorial/magazine, brutalist/raw, art deco, organic/natural, industrial/utilitarian.
- Decoration: minimal, intentional, expressive.
- Layout: grid-disciplined, creative-editorial, hybrid.
- Color: restrained, balanced, expressive.
- Motion: minimal-functional, intentional, expressive.

## Typography Defaults

- Display/hero: Satoshi, General Sans, Instrument Serif, Fraunces, Clash Grotesk, Cabinet Grotesk.
- Body: Instrument Sans, DM Sans, Source Sans 3, Geist, Plus Jakarta Sans, Outfit.
- Data/tables: Geist with tabular nums, DM Sans with tabular nums, JetBrains Mono, IBM Plex Mono.
- Code: JetBrains Mono, Fira Code, Berkeley Mono, Geist Mono.

Never recommend these as primary defaults: Inter, Roboto, Arial, Helvetica, Open Sans, Lato, Montserrat, Poppins.

Never recommend: Papyrus, Comic Sans, Lobster, Impact, Jokerman, Bleeding Cowboys, Permanent Marker, Bradley Hand, Brush Script, Hobo, Trajan, Raleway, Clash Display, Courier New for body.

## AI Slop To Avoid

- Purple/violet gradients as a default accent.
- Three-column feature grids with icon circles.
- Centered-everything layouts with uniform spacing.
- Uniform bubbly border radius on every element.
- Gradient buttons as the primary CTA pattern.
- Generic stock-photo hero sections.
- "Built for X" or "Designed for Y" copy patterns.

## Coherence Checks

- Brutalist or minimal aesthetics usually pair with minimal motion.
- Expressive color with restrained decoration makes color carry the whole mood.
- Creative-editorial layout can fight dense data; use a hybrid when needed.
- Always accept the user's final choice after naming the tradeoff.
