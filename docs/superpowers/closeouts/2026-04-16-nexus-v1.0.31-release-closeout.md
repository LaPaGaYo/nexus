## Outcome

- `v1.0.31` is the published stable patch release.
- Governed runs now leave repo-visible CCB provider and dispatch state behind,
  so provider/session failures are diagnosable from artifacts instead of only
  transient shell output.
- `/closeout` now seeds the next fresh `/discover` with an explicit bootstrap,
  reducing the continuity gap between completed work and the next run.
- Bounded dual review is faster and more stable: Codex and Gemini dispatch in
  parallel, prompts enforce a narrow fast-auditor mode, and late review replies
  can be recovered even when `ask` fails before printing a `CCB_REQ_ID`.
