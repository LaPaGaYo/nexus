# Nexus v1.0.10

Date: 2026-04-12
Tag: `v1.0.10`
Status: `release_candidate`

## Summary

This patch release focuses on governed execution clarity and user-surface
hardening.

## What Ships

- a patch-release index in `CHANGELOG.md` so incremental Nexus upgrades no
  longer look like an undocumented jump from `v1.0.0`
- smarter `CLAUDE.md` routing injection that recognizes equivalent existing
  Nexus lifecycle guidance and no longer auto-commits project files
- governed CCB handoff verification that checks every required provider in the
  requested route, not just the build generator
- per-provider governed route validation persisted into handoff artifacts so
  Codex and Gemini readiness are visible from disk instead of hidden behind one
  flat availability bit
