# Nexus v1.0.48

Date: 2026-04-27
Tag: `v1.0.48`
Status: `release_candidate`

## Summary

This release fixes legacy skill shadowing during Nexus installation. It ensures
canonical Nexus lifecycle commands cannot be silently overridden by older global
skills in Claude or shared agent skill directories.

## What Ships

- flat Claude installs now quarantine non-symlink canonical command conflicts
  before linking Nexus commands
- Codex/shared `.agents/skills` installs now quarantine legacy canonical command
  conflicts such as an old standalone `review` skill
- quarantined skills are moved to `~/.nexus/nexus-conflicts/` instead of being
  deleted
- setup regression coverage verifies both Claude and shared-agent conflict
  handling paths

- `bun test test/gen-skill-docs.test.ts test/nexus/product-surface.test.ts`

## Notes

- This specifically prevents old CCB-style `/review` skills from capturing the
  Nexus `/review` entrypoint after users choose the flat command surface.
- Existing conflicting skill directories are preserved under the Nexus conflict
  backup directory for manual recovery.
