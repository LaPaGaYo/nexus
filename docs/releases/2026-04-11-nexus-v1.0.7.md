# Nexus v1.0.7

Date: 2026-04-11
Tag: `v1.0.7`
Status: `release_candidate`

## Summary

This patch release focuses on setup correctness, clearer execution-mode
decisions, and fewer first-run surprises in the Claude install path.

## What Ships

- setup fixes that make the Claude install path ask explicitly how Nexus should
  run when CCB is present instead of silently falling back to the current local
  session
- setup fixes that ask whether to install CCB when it is missing, instead of
  quietly defaulting to a local execution path
- updated docs and product-surface checks so the install summary matches the
  actual governed-vs-local execution behavior
