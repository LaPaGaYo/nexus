# Nexus v1.0.35

Date: 2026-04-19
Tag: `v1.0.35`
Status: `release_candidate`

## Summary

This patch release fixes a governed execution gap in fresh run worktrees. Fresh
run workspaces now refresh against the remote default branch before `/handoff`
and `/build`, so bounded packets stop running against stale pre-merge baselines.

## What Ships

- fresh run worktrees now fetch the remote default branch and fast-forward
  safely before governed handoff/build dispatch when the remote has moved ahead
- if a fresh run workspace has diverged or has non-mirrored local changes that
  prevent a safe fast-forward, Nexus now blocks with an explicit workspace
  baseline error instead of letting governed execution continue on a stale repo
- handoff/build routing tests now cover both stale-before-handoff and
  stale-after-handoff remote baseline cases
