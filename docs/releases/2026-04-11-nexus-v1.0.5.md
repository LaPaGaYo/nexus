# Nexus v1.0.5

Date: 2026-04-11
Tag: `v1.0.5`
Status: `release_candidate`

## Summary

This release hardens the Claude installation path by automatically removing
legacy GSD hook wiring that can break PostToolUse during Nexus setup.

## What Ships

- a new `bin/nexus-clean-claude-hooks` helper that detects and removes legacy
  GSD `SessionStart`, `PostToolUse`, and `statusLine` commands from global
  Claude settings files
- Claude `./setup` now runs legacy hook cleanup before prompting for execution
  mode, topology, or global `CLAUDE.md` guidance
- regression coverage that locks the cleanup behavior so unrelated Claude
  settings survive while stale GSD hooks are removed

  leaves unrelated Claude configuration intact.
