# Nexus v1.0.43

Date: 2026-04-26
Tag: `v1.0.43`
Status: `release_candidate`

## Summary

This patch release fixes the post-publish maintainer refresh path in
`nexus-release-publish`. After a release tag is created, the publisher now
passes a fresh tag snapshot and live remote-release mode into
`maintainer:check`, so a successful release is no longer misreported as blocked
because the maintainer check inherited the pre-publish tag list.

## What Ships

- post-release `maintainer:check` now receives `NEXUS_EXISTING_TAGS` with the
  newly published tag included
- post-release maintainer refresh now clears stale preflight status input with
  `NEXUS_GIT_STATUS_LINES=""`
- post-release maintainer refresh now runs with `NEXUS_REMOTE_RELEASE_MODE=live`
  so the maintainer report records live remote release evidence
- release publication tests now fail if the maintainer step cannot see the tag
  that was just created
