# Nexus v1.0.38

Date: 2026-04-20
Tag: `v1.0.38`
Status: `release_candidate`

## Summary

This patch release fixes `/ship` pull-request handoff creation for governed
run worktrees.

## What Ships

- `/ship` now resolves GitHub PR head/base metadata from the governed execution
  workspace instead of the repo root
- run-owned worktrees no longer create PRs from the wrong branch direction when
  the repo root is on `main` and the actual implementation branch lives in the
  execution workspace
- ship PR handoff now stays aligned with the same workspace provenance already
  used for governed build/review/qa execution

## Notes

- This is a narrow runtime fix for `/ship` PR handoff provenance.
- It specifically closes the case where Nexus created PR metadata from the repo
  root branch instead of the active run worktree branch.
