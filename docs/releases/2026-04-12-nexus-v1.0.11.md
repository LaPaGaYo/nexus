# Nexus v1.0.11

Date: 2026-04-12
Tag: `v1.0.11`
Status: `release_candidate`

## Summary

This patch release fixes governed wrapper execution so canonical Nexus
commands keep operating against the active project repo instead of the Nexus
install root.

## What Ships

- canonical wrapper commands now capture the active repo root before changing
  into the Nexus install directory
- Nexus runtime now honors `NEXUS_PROJECT_CWD` so `/discover` through
  `/closeout` read and write lifecycle artifacts against the intended project
- generated Claude and Codex wrapper skills now pass repo cwd explicitly to
  `bin/nexus.ts`
- regression coverage that locks the wrapper contract to repo-root aware
  execution instead of install-root execution
