# Nexus v1.0.23

Date: 2026-04-13
Tag: `v1.0.23`
Status: `release_candidate`

## Summary

This patch release fixes a release-compatibility regression in the published
upgrade path. Recent `1.0.x` releases incorrectly raised
`compatibility.upgrade_from_min_version` to the immediately previous patch,
which forced users to hop through intermediate versions instead of upgrading
straight to the latest stable patch.

## What Ships

- stable patch releases now preserve direct patch-line upgrades instead of
  requiring sequential patch hops
- release preflight now blocks `release.json` manifests whose
  `compatibility.upgrade_from_min_version` exceeds the start of the current
  patch line
- `v1.0.23` restores the stable upgrade floor to `1.0.0`, so older `1.0.x`
  managed installs can upgrade directly to the latest release

  upgrades inside the `1.0.x` line.
