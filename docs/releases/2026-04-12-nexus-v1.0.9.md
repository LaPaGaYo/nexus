# Nexus v1.0.9

Date: 2026-04-12
Tag: `v1.0.9`
Status: `release_candidate`

## Summary

This patch release focuses on upgrade-time clarity and stale Claude session cleanup.

## What Ships

- clearer governed execution guidance so `governed_ccb` no longer implies that
  `primary_provider` or `provider_topology` must be configured in host state
- a new `nexus-config effective-execution` helper for inspecting the effective
  execution path without inferring it from partial config
- conservative Claude session cleanup that removes stale project transcripts
  containing obsolete Nexus prompts while preserving the newest transcript and
  nested subagent transcripts
