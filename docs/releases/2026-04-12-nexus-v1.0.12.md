# Nexus v1.0.12

Date: 2026-04-12
Tag: `v1.0.12`
Status: `release_candidate`

## Summary

This patch release tightens governed build gating so stale handoff packets are
rejected locally before any Codex dispatch starts.

## What Ships

- `/build` now validates that governed handoff artifacts record per-provider
  CCB verification for every required provider in the requested route
- stale handoff packets from older Nexus versions are blocked locally with an
  explicit rerun-`/handoff` message instead of being forwarded to Codex first
- build-stage trace artifacts now capture the local gate failure when governed
  handoff verification is incomplete
- regression coverage locks the “no dispatch on stale governed handoff” path
