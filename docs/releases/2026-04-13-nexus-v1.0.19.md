# Nexus v1.0.19

Date: 2026-04-13
Tag: `v1.0.19`
Status: `release_candidate`

## Summary

This patch release closes a closeout-history bug exposed by a real governed
discover-to-ship run. Nexus previously accepted closeout after `plan -> ...`
flows but rejected the same run once it included the canonical `discover` and
`frame` stages up front.

## What Ships

- `/closeout` now accepts the real full governed lifecycle prefix:
  `discover -> frame -> plan -> handoff -> ...`
- full discover-to-ship governed runs no longer fail with
  `Illegal Nexus transition history` after a valid `/ship`
- existing fix-cycle, review-retry, QA, and ship closeout validation remains
  unchanged
