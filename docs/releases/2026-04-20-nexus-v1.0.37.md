# Nexus v1.0.37

Date: 2026-04-20
Tag: `v1.0.37`
Status: `release_candidate`

## Summary

This patch release fixes a governed QA fix-cycle provenance bug that could
poison `/build` and then false-block `/review` with `Requested and actual audit
route diverged`.

## What Ships

- QA-driven fix-cycle `/build` no longer inherits `qaStatus.requested_route`
  into build artifacts; it now preserves canonical build routing and evaluator
  provenance
- `/review` now falls back to canonical build or handoff routing when older
  malformed build artifacts would otherwise make requested audit route
  derivation impossible
- QA fix-cycle runs can now re-enter `/review` and `/qa` cleanly instead of
  getting trapped behind poisoned build route metadata
