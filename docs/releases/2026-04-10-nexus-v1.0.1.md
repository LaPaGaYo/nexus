# Nexus v1.0.1

Date: 2026-04-10
Tag: `v1.0.1`
Status: `ready_to_publish`

## Summary

This release turns the release-based upgrade model into a maintainer-grade
published flow.

## What Ships

- release-based `nexus-update-check` with managed install metadata fallback
- `nexus-upgrade-install` for published release bundles
- `nexus-release-preflight` for local maintainer validation before tagging
- `nexus-release-smoke` for post-publish remote release verification
- conservative legacy migration and vendored sync behavior for managed installs
