# Nexus v1.0.6

Date: 2026-04-11
Tag: `v1.0.6`
Status: `release_candidate`

## Summary

This patch release focuses on first-run stability, install-surface cleanup, and
more predictable Nexus guidance during setup and skill invocation.

## What Ships

- fixes to the generated Nexus preamble so first-use guidance stays Nexus-owned
  and no longer opens unexpected external references
- install and support-surface cleanup that removes deprecated runtime helpers
  and reduces stale compatibility behavior in fresh installs
- updated docs and generated skill surfaces so the active Nexus contract is
  cleaner and easier to reason about during setup, upgrades, and daily use
